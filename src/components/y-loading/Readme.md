## 动画加载

### 如何使用
```js
import YLoading from '@/components/y-loading/index.vue'
```
```html
<y-loading :yLoading.sync='loadingStatus'></y-loading>
```

### Props

| name          | 描述                               | 默认值 |
| ------------- | ---------------------------------- | ------ |
| yLoading      | 是否加载                              | true     |
             |                                   |      |
